2. In `script.js`, replace `YOUR_WEB3FORMS_ACCESS_KEY`
3. Uncomment the fetch code

**Option B: EmailJS**
1. Sign up at https://www.emailjs.com
2. Add EmailJS SDK to contact.html
3. Update form submission code
